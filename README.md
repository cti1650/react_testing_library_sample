# React Testing Library Sample

## インストール

```json: package.json
{
  "script": {
    ...
    "test": "jest --watch"
    ...
  }
}
```

```bash
yarn add -D @testing-library/dom @testing-library/jest-dom @testing-library/react @testing-library/user-event @types/jest babel-jest jest jest-css-modules jest-environment-jsdom
```
